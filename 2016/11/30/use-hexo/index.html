使用Hexo

必备软件

  • nodejs（根据平台选择安装包即可）
  • git（同上）
  • hexo（是js脚本程序 npm install -g hexo）
  • hexo-deployer-git（如果用 git 部署的话，需要这个包。npm install --save hexo-deployer-git）

使用

安装好 hexo 包之后，打开 git bash 命令行窗口，可以和 hexo 交互了。
