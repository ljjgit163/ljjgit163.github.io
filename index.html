使用Hexo博客系统有感

Hexo 是一个静态网站生成器。原理是用程序自动读取某些文件的内容，然后把这些内容填充到模板里面去。

Hexo 是一个 nodejs 的项目。据说是一个台湾小伙子写的。

使用 Hexo 主要是用它提供的命令行接口来操作的。

例如：

Hexo 支持 Markdown 格式书写文章。这使得写文章写起来很舒服，不会再被样式所困惑，让作者更多地回归到内容上。

Hexo 支持很多皮肤，和模板，但是这要建立在使用者足够熟悉这套系统上，再去完它们。
